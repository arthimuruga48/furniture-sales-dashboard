# furniture-sales-dashboard
This project presents a comprehensive sales analysis for a furniture business using Power BI. The dashboard provides insights into sales performance, product categories, customer purchasing trends, and overall business growth.
Project Objectives
Analyze furniture sales data.
Identify top-performing products and categories.
Monitor sales trends over time.
Track key business metrics using interactive visualizations.
Support data-driven decision making.
Tools Used
Power BI – Data visualization and dashboard creation
Power Query – Data cleaning and transformation
SQL – Data analysis and querying
Excel/CSV – Data source
Dashboard Features
Total Sales KPI
Total Orders KPI
Product Category Analysis
Top Selling Furniture Products
Monthly Sales Trend Analysis
Interactive Filters (Slicers)
Dynamic Visualizations and Reports
Key Insights
Identified the highest revenue-generating furniture products.
Analyzed category-wise sales performance.
Observed monthly sales patterns and trends.
Enabled quick business performance monitoring through interactive dashboards.
